# E-Commerce User Service

Spring Boot project for managing users in an e-commerce platform.

## Features
- User registration & login
- Password encryption (BCrypt)
- JWT-based authentication
- Secured endpoints for profile access

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- MySQL
- JWT

## How to Run
1. Clone the repo
2. Configure `application.properties` for your database
3. Run `mvn spring-boot:run`
